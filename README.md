# Car Rental Management System (Bash)

A Linux based CLI car rental management system.

## Features

- CRUD operations
- Modular Bash scripts
- Logging system
- Backup and restore
- Cron automation
- Remote backup using SSH + rsync
- File locking using flock

## Technologies

- Bash
- Linux
- awk
- sed
- cron
- rsync
- SSH
- Git

## Run

chmod +x main.sh

./main.sh
